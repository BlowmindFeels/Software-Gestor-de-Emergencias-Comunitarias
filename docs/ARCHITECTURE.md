# Arquitectura 3 capas (3 tiers)

- Presentación (Frontend): React + Vite. Interfaz de usuario, mapas y componentes en tiempo real.
- Lógica / Aplicación (Backend): Node.js + Express. APIs REST para gestión de emergencias y lógica de negocio.
- Datos (Persistencia): Base de datos (Mongo/Postgres) accesible desde `backend/src/config/db.js`.

Cada carpeta contiene archivos base para empezar el desarrollo.
